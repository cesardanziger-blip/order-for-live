# OrderForLive

Projeto Full Stack para fins de teste desenvolvido com .NET 10 e React.

### Backend
- .NET 10
- Entity Framework Core
- SQLite
- Swagger

### Frontend
- React
- TypeScript
- Vite

### Infraestrutura
- Docker
- Docker Compose

### Funcionalidades
Cadastro de pedidos
Listagem de pedidos
Edição de pedidos
Exclusão de pedidos

### Execução
docker compose up --build

### URLs
Frontend:
http://localhost:3000

Swagger:
http://localhost:5000/swagger
